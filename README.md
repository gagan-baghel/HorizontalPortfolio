# Portfolio Horizontal (Next.js)

![App Screenshot](./public/appScreenshot.png)

A horizontal-scrolling personal portfolio built with Next.js, React, Tailwind CSS, GSAP, and Framer Motion.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Lint

```bash
npm run lint
```
